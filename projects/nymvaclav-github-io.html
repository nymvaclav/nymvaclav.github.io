About this project

This is a simple website hosted on GitHub Pages designed to represent me. I can easily point to all projects I have finished or projects I am still working on thanks to it!

In the future I'd like to...

In the future I would like to move this website off of GitHub and host it on a dedicated webserver with my own domain. Technically I could do that right now and be done with it in a moment's notice, but I don't want to. This website is written in HTML and CSS, thus everytime I want to add a new entry I have to create a new HTML document.

What I want to do is to create a Django Web Server, and host this website using Django. Then I could write each entry into a databse and dynamically render it. This approach has no real game-changing advantage (it's just more resource-hungry :), but it looks better and it's wonderful for scaling this website.

I could then create my own blog, for example, or write tutorials on how to learn different programming languages, which is what I actually want to do — encourage people to learn how to code and create amazing projects. Thank you for reading this, as just a few people interested in my journey is all it takes to build wonderful things!
