About this project

This is an application with a Graphical User Interface designed to convert multiple filetypes at once. If you have a folder with hundreds of images and you need to convert them to .png, for example, you can use this application to do it. You can convert multiple .webp, .png and .jpg images at once, and locally! This means the you do not need to upload your sensitive images to a remote server. This application is also customizable, as you can select you preferred theme. To use it, clone the nymvaclav/image-filetype-convertor repository into your computer, install required dependencies, and run the convertor.py file.
