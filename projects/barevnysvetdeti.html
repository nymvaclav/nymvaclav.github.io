About this project

Barevný svět dětí is a nonprofit organization focused on supporting children under the care of OSPOD. This web application is used to track how much time assistents spend with clients assigned to them by calculating the time spent on each meeting. This data is then used to calculate their salary, for example. All accounts are associated with an email address, where emails about successfull logins, new clients being assigned to them, or their accounts removal are being sent to via Django’s email-sending API. There is much more to talk about, but I can not share that at this time to maintain confidentiality.

This project's goal

Developed using Django, this project prioritizes efficiency, data security, and ease of use for its users. The goal is to create a reliable, scalable platform that will work for many years to come. This project is being developed by me alone and will be accessible to this non-profit's employees only. If Barevný svět dětí allows me to do so, I will release parts of the source code into a repository on my GitHub profile.
