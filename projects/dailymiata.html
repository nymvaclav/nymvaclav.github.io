About this project

This is a Discord bot that is designed to send a picture of a NA Miata every day into all #dailymiata Discord channels the bot is in. I made it to learn how to create Discord bots, but honestly, I love it! I also programmed it to use Limewire API to generate an image of a Miata every day using Artificial Inteligence. As of today (8/2/2025) I have abandoned this project, but if I find some free time in the future I might get back to it. It is currently NOT hosted on any servers, thus the bot offline. If I get back to it I will make sure to put the invite link on this page.
